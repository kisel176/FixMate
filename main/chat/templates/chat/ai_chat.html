Ваша функция работает, но имеет сложность O(n²). Вот оптимизированная версия:

def find_duplicates(arr):
    seen = set()
    duplicates = set()
    for item in arr:
        if item in seen:
            duplicates.add(item)
        else:
            seen.add(item)
    return list(duplicates)

✅ Сложность уменьшена до O(n)

✅ Используется множество для поиска дубликатов

✅ Код стал более читаемым
